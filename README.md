## Fizz demo

`packages/fizz`

### Setup

1. Install dependencies using pnpm:
   ```bash
   pnpm install
   ```

2. Build the project:
   ```bash
   pnpm build
   ```

3. Start the server:
   ```bash
   pnpm start
   ```
